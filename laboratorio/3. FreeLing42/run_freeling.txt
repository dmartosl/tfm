FreeLing
--------

$ wget https://github.com/TALP-UPC/FreeLing/releases/download/4.2/FreeLing-langs-src-4.2.tar.
$ tar -xzvf FreeLing-src-4.2.tar.gz
$ cd FreeLing-4.2/
$ mkdir build
$ cd build/
$ cmake .. -DPYTHON3_API=ON -DWARNINGS=OFF
$ make install
$ cd /usr/local/share/freeling/APIs/python3
$ python3 sample.py < /home/uned/Documents/ejemplo.txt
$ analyze -f es.cfg < /home/uned/Documents/ejemplo.txt
